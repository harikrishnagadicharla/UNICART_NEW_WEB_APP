# UniCart

Modern e-commerce platform built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Pages and route components
- `/components` - Reusable UI elements
- `/store` - Zustand stores (cart, auth, wishlist)
- `/lib` - Utilities (formatters, validators, helpers)
- `/data` - Mock JSON data
- `/public` - Static assets

## Features

- Modern, responsive design with generous spacing
- Professional layout with proper hierarchy
- Category browsing
- Featured products section
- Newsletter signup
- Clean, maintainable code structure
